7. Moving Forward

7.1 Project Deliverable and Scope

As discussed with the team, the project deliverable is a fully functioning robomaster robot by the end of the project timeline. Therefore the detailed scope will be set as “ Develop a fully functioning step climbing robot with mechanically optimized wheel leg structure, dynamically motion controlled and well tuned robot for Robomaster competition by the end of the project.”
